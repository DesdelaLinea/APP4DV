Falta conectar Firebase

Este archivo necesita las credenciales de tu proyecto de Firebase para poder guardar clubes, miembros y oportunidades.

  1. Entra a console.firebase.google.com y crea un proyecto (o usa uno existente).
  2. Activa Firestore Database.
  3. Ve a Configuración del proyecto → General → "Tus apps" → crea una app web y copia el objeto firebaseConfig.
  4. Pégalo dentro de este archivo, reemplazando el bloque PEGA_AQUI_TU_... cerca del final.

Un paso nuevo: activar Autenticación Anónima

Esta versión usa autenticación real (no solo el navegador) para que los roles y permisos no se puedan falsificar. Ve a Authentication → Sign-in method y activa el proveedor Anónimo. No pide correo ni contraseña al jugador, solo genera una identidad segura por dispositivo.

Y sube las reglas de seguridad

Copia el archivo firestore.rules que te compartí junto a este HTML, y pégalo en Firestore Database → Reglas en la consola de Firebase. Sin esto, los permisos de club/red no se hacen cumplir de verdad.
